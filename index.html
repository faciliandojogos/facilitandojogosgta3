Versões dubladas e traduzidas com e sem mod menu Cleo.

Sobre o GTA 3 para Android

GTA 3 é um clássico da Rockstar adaptado para Android, mantendo a essência do jogo original. Apesar de ser uma ótima adaptação, muitos jogadores sentem falta de melhorias na interface e na jogabilidade.

Versão com Mod Cleo + Dublagem

Essa versão traz o GTA 3 com dublagem em português BR e mod menu Cleo incluso, permitindo acessar diversas funções como: vida infinita, dinheiro infinito, carros, armas, entre outros. Atenção: essa versão NÃO possui interface em português, apenas a dublagem.

Versão com Interface PT-BR + Dublagem

Essa é a versão ideal para quem quer jogar com tudo traduzido. Ela conta com a interface completa em português do Brasil, além da dublagem. Porém, não possui o mod menu Cleo incluso.
